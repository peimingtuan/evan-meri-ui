### 介绍

Meri Plus 是一个Vue3 的组件库，提供开箱即用的高质量 Vue 组件。它提炼自企业级中后台产品的交互语言和视觉风格，开发和服务于企业级后台产品。

要了解如何安装，参见`快速上手`

### 特性

- 🚀 性能极佳，组件平均体积小于 1KB（min+gzip）
- 🚀 30+ 个高质量组件，覆盖PC端主流场景
- 💪 使用 TypeScript 编写，提供完整的类型定义
- 📖 提供 figma 设计资源
- 🍭 支持 Vue 2、Vue 3 
- 🍭 支持主题定制，内置主题变量
- 🍭 支持按需引入和 Tree Shaking
### 快速上手
请阅读`快速上手`章节，通过该章节你可以了解到meri Plus的安装方法和基本使用姿势。
